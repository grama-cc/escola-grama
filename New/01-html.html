HTML

HTML é a sigla de HyperText Markup Language, "Linguagem de Marcação de Hipertexto". Consiste em uma linguagem de marcação utilizada para produção de páginas na web, que permite a criação de documentos que podem ser lidos em browsers (Google Chrome, internet explorer etc) e transmitidos pela internet. Onde deve ficar o conteudo do site.

A coisa mais importante quando queremos começar um site é saber qual vai ser o conteúdo dele

Usamos as "Heading tags" para definir a Hierarquia dos nossos conteudos. Dentro da tag h1 a gente coloca os titulos mais importantes, como por exemplo: "Guia da pizza perfeita". Mais tarde, usaremos o css para definir tamanho, cor, tipografia entre outras caracteristicas que desejemos dar a esses títulos.

Dentro de h2 colocamos títulos menos importantes que, em h1, ou subtitulos
A gente pode continuar descendo, h4, h5, h6... até não ter importância quase nenhuma.

Para escrevermos pagragrafos, anotações ou frases devemos coloca-las na tag "p".

Farinha delicia

Podemos colocar mais de uma tag em um conteudo, aqui colocamos uma "a" dentro de uma "h1". "a" significa "anchor" ou âncora. Para que a âncora funcione você tem que dizer para onde ela deve ser direcionada, ou seja o "href". Usamos o target="_blank" para que o link abra em uma nova janela. O default(o que acontece caso não especifiquemos o traguet) é target="_self", o que faz com que a página abra na mesma janela. Temos dois tipos de href:

  • para fora do nosso site (absolute path)
  • para dentro (relative path).

Quando estamos programando um site fazemos uma pasta e colocamos cada seção dele em um arquivo .html diferente. O ideal é nomea-los com o nome da seção que representam como por exemplo: "contato.html" se referindo a página "contato" do site

O "a" é uma tag com o display inline, isso quer dizer que um link entra do lado de outros elementos, o "a" não tenta acupar toda a largura da tela como uma tag "p", "ul", "li" entre outras que são tags com display block.

A tag "span" também é uma tag inline, usamos essa tag para marcar partes espeficicas do nosso texto e dar a elas um estilo diferente usando o css.

Uma outra tag muito utilizada para demarcar elementos é a "div", que é uma divisão genérica do cógigo. Nas "div"s nós podemos ter todo tipo de elementos.
