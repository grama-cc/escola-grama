Imagens

Existem 3 tipos de imagens que podem aparecer no nosso site

  • Content images - imagens de conteudo são imagens que são tão nescessárias quando o texto.

    Ex: Em um site que se propõe a vender roupas, a foto da peça ao lado do preço é tão importante quanto o preço

  • Layout images - imagens que estão no layout mas que não são nescesárias para o entendimento do conteudo de uma página.

    Ex: A imagem usada no fundo de uma página

  • User interface images - imagens que são usadas para ajudar na interface da página, mas não são nescessárias para o funcionamento da página

    Ex: um bom exemplo disso são as "setinhas" ao lado dos itens do menu

As imagens devem ser colocadas em uma pasta separada dentro da nossa pasta principal de projeto. Por isso o "src", neste caso, ficaria algo assim "images/nomedasuaimagem.png"

Mas onde colocamos essas iamgens no nosso código?

  • Content images - já que essas imagens são parte do conteudo do site, as colocamos dentro do html
  • Layout images - já que essas imagens fazem parte do estilo do site devemos coloca-las no css
  • User interface images - já que essas imagens fazem parte da interface do site, ou seja, do estilo da nossa página, devemos coloca-las no css.
